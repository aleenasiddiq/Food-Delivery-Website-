### 1. Menu Management (Array & Bubble Sort)
- **File:** `classes/ArrayMenu.php`, `classes/BubbleSort.php`
- **Concept:** All menu items are fetched into an **Array**.
- **Sorting:** We implemented **Bubble Sort** (`BubbleSort::sort()`) to manually sort the menu array by **Price** or **Rating** without using built-in PHP sort functions (like `usort`).

### 2. Cart System (Stack)
- **File:** `classes/StackCart.php`
- **Concept:** The cart follows the **LIFO (Last In, First Out)** principle using a **Stack**.
- **Operations:**
    - `push($item)`: Adds an item to the cart.
    - `pop()`: Removes the *last added* item (Undo feature).

### 3. Order Processing (Queue)
- **File:** `classes/QueueOrder.php`
- **Concept:** Orders are processed using the **FIFO (First In, First Out)** principle.
- **Operations:**
    - `enqueue($order)`: Customer places an order (added to the back of the queue).
    - `dequeue()`: Admin processes/serves the order (removes from the front of the queue).
